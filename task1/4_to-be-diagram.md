# TO-BE Диаграмма автоматизированного процесса производства контента

```mermaid
sequenceDiagram
    %% PARTICIPANTS
    participant SMM as SMM-менеджер
    participant EXP as Эксперт
    participant COPY as Копирайтер
    participant DSG as Дизайнер
    participant VID as Монтажёр
    participant ANA as Аналитик
    participant AI_PM as AI-оркестратор контента
    participant AI_CPY as AI-копирайтер
    participant AI_DSG as AI-дизайн агент
    participant AI_VID as AI-видео агент
    participant AI_ANA as AI-аналитик

    %% ИНИЦИАЦИЯ ПРОИЗВОДСТВА

    SMM->>AI_PM: Инициация спринта контент-плана
    note right of AI_PM: 
        Вход: контент-план, инфоповоды<br>
        Выход: авто-сгенерированный список задач<br>
        Метрика: полнота покрытий<br>
        Системы: Notion API, Sheets API<br>
        Автоматизация: №1 (автоформирование задач)

    AI_PM-->>SMM: Готовый бэклог задач

    %% СБОР ДАННЫХ ОТ ЭКСПЕРТА

    AI_PM->>EXP: Авто-запрос исходной информации
    note right of AI_PM:
        Вход: темы<br>
        Выход: запросы + напоминания<br>
        Метрика: скорость получения данных<br>
        Системы: Telegram Bot API<br>
        Автоматизация: №2 (авто-запросы)
    
    EXP-->>AI_PM: Загрузка материалов через форму/бота
    note right of EXP:
        Вход: форма<br>
        Выход: структурированные материалы<br>
        Метрика: полнота данных<br>
        Системы: Web-form, GDrive API<br>
        Автоматизация: №3 (структурированный сбор)

    AI_PM-->>SMM: Готовый бриф (авто-сбор данных)
    note right of AI_PM:
        Автоматизация №4 (авто-генерация брифа)

    %% ТЕКСТЫ

    AI_PM->>AI_CPY: Генерация текстового черновика
    note right of AI_CPY:
        Вход: бриф<br>
        Выход: 2–3 варианта текстов<br>
        Метрика: качество текста, тональность<br>
        Системы: GPT API<br>
        Автоматизация: №5 (ИИ-черновики)

    AI_CPY-->>SMM: Авто-доставка черновиков
    note right of AI_CPY:
        Автоматизация №6 (маршрутизация)

    SMM->>COPY: (Опционально) ручная доработка
    note right of COPY:
        Вход: авто-черновики<br>
        Выход: финализированные тексты<br>
        Системы: Docs<br>
        Метрика: глубина смысла

    %% ВИЗУАЛ

    AI_PM->>AI_DSG: Авто-ТЗ для визуала
    note right of AI_PM:
        Вход: финальный текст<br>
        Выход: краткое ТЗ + референсы<br>
        Автоматизация: №7 (авто-ТЗ)

    AI_DSG->>DSG: Генерация набора концептов
    note right of AI_DSG:
        Вход: ТЗ<br>
        Выход: 3 варианта визуала<br>
        Метрика: бренд-соответствие<br>
        Системы: Midjourney API, Figma API<br>
        Автоматизация: №8

    DSG-->>AI_PM: Финальный визуал
    note right of DSG:
        Автоматизация №9 (авто-выгрузка)

    %% ВИДЕО

    AI_PM->>AI_VID: Авто-сбор материалов для монтажа
    note right of AI_PM:
        Вход: текст + визуал<br>
        Выход: структурированный пакет<br>
        Автоматизация №10

    AI_VID->>VID: Авто-монтаж первоначального видео
    note right of AI_VID:
        Вход: материалы<br>
        Выход: черновик видео<br>
        Метрика: retention, динамика<br>
        Системы: Runway API, CapCut API<br>
        Автоматизация: №11

    VID-->>AI_PM: Черновик видео
    note right of VID:
        Автоматизация №12 (авто-загрузка)

    %% СОГЛАСОВАНИЕ С ЭКСПЕРТОМ

    AI_PM->>EXP: Авто-согласование контента
    note right of AI_PM:
        Вход: текст+визуал+видео<br>
        Выход: комментарии → задачи<br>
        Системы: Docs/Comments API<br>
        Автоматизация: №13

    EXP-->>AI_PM: Подтверждение / правки

    %% ПУБЛИКАЦИЯ

    AI_PM->>SMM: Автосборка поста
    note right of AI_PM:
        Вход: текст, визуал, хэштеги<br>
        Выход: собранный пост<br>
        Автоматизация: №14

    AI_PM->>SMM: Автопостинг по расписанию
    note right of AI_PM:
        Системы: Meta API, TikTok API<br>
        Автоматизация: №15

    %% АНАЛИТИКА

    AI_PM->>AI_ANA: Автосбор статистики
    note right of AI_ANA:
        Вход: соцсети API<br>
        Выход: сырые данные<br>
        Автоматизация: №16 + №17 (API-интеграции)

    AI_ANA->>ANA: Готовый авто-отчёт
    note right of AI_ANA:
        Вход: метрики<br>
        Выход: отчёт + интерпретации<br>
        Автоматизация: №18

    ANA-->>AI_PM: Рекомендации

    %% ОБНОВЛЕНИЕ ПЛАНА

    AI_PM->>SMM: Автогенерация новых тем
    note right of AI_PM:
        Вход: отчёт<br>
        Выход: идеи, темы, приоритеты<br>
        Автоматизация: №19

    SMM->>SMM: Финальная корректировка плана
```
